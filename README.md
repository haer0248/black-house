# Step

1. Go to [Discord Developer Portal](https://discord.com/developers/applications) create bot.
2. Rename `.env.sample` to `.env`.
3. Enter require Discord bot data.
4. `npm start`.

# Permission

- Moderate Members
- Send Message

# Scopes

- applications.commands
- bot
